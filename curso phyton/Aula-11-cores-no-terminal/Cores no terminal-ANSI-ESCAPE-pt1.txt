   Usando ANSI escape sequence.

Para mostrar uma cor em python use o comando: \033[style text back  m. no meio coleque o estilo, text e back(background)  

exemplo: \033[0;33;44m

TIPOS DE STYLE
0 = NONE
1 = NEGRITO
4 = UNDERLINE - SUBLINHADO
7 = NEGATIVE

CORES DE TEXT:
30 = BRANCO
31 = VERMELHO
32 = VERDE
33 = AMARELO
34 = AZUL
35 = ROXO
36 = AZUL MARINHO
37 = CINZA

CORES DE BACK
40 = BRANCO
41 = VERMELHO
42 = VERDE
43 = AMARELHO
44 = AZUL
45 = ROXO
46 = AZUL MARINHO
47 = CINZA

Para inverter as core normal use: \033[7:30m
