Para fazer uma variavel em phyton use o sinal de atribuição =

exemplo :

nome = 'Doca'
idade = 27
peso = 120

Use print com virgulas no parenteses para mostrar as variaveis.

print (nome, idade, peso )

Resultado = Doca 27 120


virgula, é usado para mostrar mensagens e números.

Use input('escreva a pargunta aqui') atribuindo a variavel para permitir o usuario que escreva a variavel.

exemplo :

nome = input('Qual é o seu nome? ')
Qual é o seu nome? doca  <-- vai aparecer dessa forma no progama e em seguida coloque a resposta ao lado.

nome

Resultado : doca

exemplo:

nome = input('Qual é o seu nome? ')

--Criando scripts em phyton--

crie uma pasta  e  va em IDLLE new file

e coloque os comando nesse file para salvar scripts.

nome = input('Qual é o seu nome? ')
idade = input('Qual é a sua idade? ')
peso = input('Qual é o seu peso? ')
print(nome, idade, peso)

depois abra o arquivo em no IDDLE SHELL

e clique em Run Module

exemplo de script:

nome = input('Qual é o seu nome? ')
idade = input('Qual é a sua idade? ')
peso = input('Qual é o seu peso? ')
print('Seu nome é:',nome)
print('Sua idade é:',idade)
print('Seu peso é:',peso)
