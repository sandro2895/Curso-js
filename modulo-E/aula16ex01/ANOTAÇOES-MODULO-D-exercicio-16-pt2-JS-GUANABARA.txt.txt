para adicionar algo ao resultado de for ou alguma outra coisa
use += concatenação apos a chaves com (VARIAVEL).innerHTML.

EXEMPLO:

for(let c = i; c <= f; c += p) 

    res.innerHTML += ` ${c} \u{1F449}`

  }
  res.innerHTML += `\u{1F3C1}`
}
